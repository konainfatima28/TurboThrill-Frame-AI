# Contributing

Thank you for your interest in contributing to TurboThrill Frame AI!

## Getting Started

1. Fork the repository.
2. Clone your fork:
   ```bash
   git clone https://github.com/yourusername/TurboThrill-Frame-AI.git
   ```
3. Create a new branch:
   ```bash
   git checkout -b feature/your-feature
   ```
4. Make your changes.
5. Commit:
   ```bash
   git commit -m "Add: short description"
   ```
6. Push your branch:
   ```bash
   git push origin feature/your-feature
   ```
7. Open a Pull Request.

## Code Style

- Follow PEP 8.
- Write clear, modular code.
- Add comments where necessary.
- Update documentation if behavior changes.

## Reporting Issues

Please use the GitHub Issue templates when reporting bugs or requesting features.
