# Code of Conduct

## Our Pledge

We are committed to fostering an open, welcoming, and respectful community.

## Expected Behavior

- Be respectful and professional.
- Welcome constructive feedback.
- Help others learn and grow.
- Focus on improving the project.

## Unacceptable Behavior

- Harassment or discrimination.
- Personal attacks.
- Spam or abusive language.
- Any behavior that creates an unwelcoming environment.

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this Code of Conduct.
